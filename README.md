# Rag ComplyAI

Rag ComplyAI is a modern Next.js web app for secure, AI-driven analytics and management of regulatory and FDA compliance documents.

## Features

- Fast, secure login & registration
- AI-powered document insights
- Easy file uploads
- ChatBot for compliance questions

## Getting Started

1. Install dependencies:

   ```bash
   bun install
   ```

2. Run the development server:

   ```bash
   bun run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` — Main app pages and layouts
- `components/` — UI components (NavBar, ChatBot, etc.)
- `lib/` — Hooks and utilities

## Tech Stack

- Next.js (App Router)
- React
- Typescript
- Resend
- React Email
- Neon (Postgress host)
